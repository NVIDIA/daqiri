DAQIRI Connects Sensor Data to the NVIDIA Compute Ecosystem

DAQIRI (Data Acquisition for Integrated Real-time Instruments) moves high-bandwidth data between external sensors and GPU, CPU, or storage devices. Today, streams arrive from network-capable sensors over raw Ethernet, Linux TCP/UDP sockets, or RoCE/RDMA, giving applications one zero-copy path for ingest and egress; ingest from programmable PCIe devices such as FPGAs is on the roadmap. DAQIRI not only accelerates data movement and storage at the instrument but can also connect sensor data to HPC and Cloud systems.
